# Customer Base AuditThis project analyzes customer activity and profitability using customer transaction data. The dataset was cleaned and organized in Google Sheets to prepare it for analysis.Key metrics such as number of customers, transactions, revenue, and profit were examined across different quarters and customer cohorts. Linear charts were used to visualize trends and better understand customer behavior over time.## Tools- Google Sheets## Skills Demonstrated- Data cleaning and preparation- Tabulation and data organization- Cohort analysis- Data visualization- Spreadsheet analysis
